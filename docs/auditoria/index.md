## Enfoque de la auditoría

La auditoría se plantea como una revisión **documental, técnica y metodológica**. No busca modificar el código ni certificar oficialmente el sistema, sino verificar si el proceso de desarrollo se encuentra suficientemente sustentado con evidencias.

<div class="defense-box" markdown>

**Idea para exposición:**  
“La auditoría SDLC permite revisar si el proyecto All-InOne fue planificado, desarrollado, probado y documentado de forma coherente, usando como base el Project Charter, el checklist, las evidencias técnicas y los estándares de referencia.”
