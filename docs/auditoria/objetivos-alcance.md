# Objetivos y alcance

## Objetivo general

Realizar una auditoría formal del Ciclo de Vida del Desarrollo de Software del proyecto **All-InOne**, evaluando procesos, documentación, arquitectura, gestión, implementación, pruebas, seguridad y entregables para determinar su grado de cumplimiento frente a estándares y buenas prácticas SDLC.

## Objetivos específicos

| Código | Área evaluada | Qué se revisa |
|---|---|---|
| OE1 | Metodología y gestión | Planificación, Jira, sprints, responsables, criterios de aceptación y seguimiento. |
| OE2 | Requisitos | Historias de usuario, requisitos funcionales/no funcionales y trazabilidad. |
| OE3 | Diseño | Arquitectura, módulos, capas, multitenancy y coherencia con implementación. |
| OE4 | Construcción | Buenas prácticas, stack tecnológico, control de versiones y organización modular. |
| OE5 | Pruebas | Pruebas funcionales, E2E, seguridad, rendimiento y evidencias de resultados. |
| OE6 | Implementación | Instalación, configuración, despliegue de prueba, Swagger, Jenkins y DevSecOps. |
| OE7 | Mantenimiento | Modularidad, cambios, defectos, auditoría, soft delete y extensibilidad. |
| OE8 | Documentación | Informe, Jira, pruebas, endpoints, anexos, manuales y evidencias. |
| OE9 | Seguridad | JWT, roles, permisos, rutas protegidas, validaciones y aislamiento por tenant. |
| OE10 | Calidad | ISO/IEC 25010: funcionalidad, fiabilidad, usabilidad, seguridad, mantenibilidad y portabilidad. |

## Exclusiones principales

La auditoría no incluye:

- auditoría financiera o comercial;
- revisión legal de contratos o licencias;
- auditoría de infraestructura física;
- certificación ISO formal;
- revisión línea por línea de todo el código;
- corrección directa de errores;
- implementación de módulos futuros;
- pruebas de penetración avanzadas;
- garantía de disponibilidad productiva.

<div class="defense-box" markdown>

**Idea clave:**  
“El alcance define qué se revisa; las exclusiones aclaran qué no forma parte de la auditoría para evitar prometer más de lo que realmente se evaluará.”
