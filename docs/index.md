## ¿Por qué están juntas estas dos secciones?

La documentación frontend responde **cómo se construye la experiencia visual y la integración con la API**.  
La documentación de auditoría responde **cómo se evaluó el ciclo de vida del desarrollo del proyecto All-InOne**.

<div class="audit-callout" markdown>

**Lectura correcta del sitio:** el frontend muestra evidencia técnica de interfaz, rutas, componentes y pruebas; la auditoría SDLC organiza esa evidencia dentro de una evaluación formal.
